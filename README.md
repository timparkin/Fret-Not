# Fret-Not

run using 

python -m <module>
	
The module names available are  


caged - produces caged shapes (x2 diagrams)

noteboard - produces a standard fretboard with notes

intervals - produce a set of fretboards showing intervals

major_notes - marks all the triads for A

triads - produces separate triads and inversions 

ORM - produces string skipping triads in R, 3, 5 order

7s - produces seventh shapes

changes - produces a set of chord changes
